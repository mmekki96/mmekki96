### 📂 Featured Projects

#### 🌡️ [PIC16-Thermosense](https://github.com/mmekki96/PIC16-Thermosense)
A modular digital thermometer using the PIC16F877A and LM35 sensor. Focuses on clean, modular C code (XC8) and precise ADC interfacing displayed on a 16x2 LCD.

#### 🕹️ [GPIO-Simulator-CLI](https://github.com/mmekki96/GPIO-Simulator-CLI)
A CLI-based tool written in C to simulate General Purpose Input/Output behavior, useful for testing logic before deploying to hardware.

#### ⚡ [Hello_world-STM32-Bare-metal](https://github.com/mmekki96/Hello_world-STM32-Bare-metal)
A foundational project demonstrating how to bring up an STM32 board from scratch without heavy abstraction layers, focusing on register-level programming.
